<!-- node 2026.01.01 -->
今天是2026的第一天，未来可期！  
元旦起来就是收拾打包行李，收衣服，拆架子，装电器。

<!-- node 2026.01.02 -->
昨天还没完全收拾打包好，早上还得收拾。  
中午樱同学姐来到，刚好这时候也收拾好东西了，遂一起把包裹都搬到一楼。  
一开始叫了一个中等面包车过来，除去电动车，其他行李刚刚好装得下，最后再叫了一个微面单独载电动车过来。  
我跟车第一辆装行李的货拉拉先到，把行李都卸下后，中介已经到了，上去跟她对了家具清单和移交钥匙。
第二辆装电动车的货拉拉即到。这时候樱同学和她姐一起开车过来，还没到呢。我先开始搬运上楼。  
行李得一个个搬到四楼步梯，着实累。  
三个人一起都搬上去之后，业主就过来了，再签署了一个合同，他用来交税的。  
五点开车去附近的龙泉山庄吃饭，烧鸡+乳鸽，不太好吃。  
晚上十点池湘和她男朋友从汕头开车到我这，带来了牛肉丸和蛋糕，煮汤圆糖水。

<!-- node 2026.01.03 -->
行李包装解压安装。  
宽带移机，网络恢复。  
上班地铁路线探查。  
化龙镇上熟悉街道。

<!-- node 2026.01.04 -->
后续需要寻找附近的娱乐项目：
- 野餐露营
- 野钓
要买的东西：
- 摇椅

樱同学新年第一天上班就出差，很突然，早上若无其事上班，下午就出发安徽黄山。

<!-- node 2026.01.07 -->
今天听闻前公司appeon的sd组在商谈裁员解散，n+1。  
不后悔6月份就裸辞，毕竟对现在的处境很满意，只恨不早个一年半载解散。  
"人生如棋，落子无悔"

<!-- node 2026.01.09 周五 --> 
早上八点多起床，爸就打电话过来，说今天上来广州，电话那头他们还在卖鱼。匆忙买了标致11.50的大巴车票上来。  
早上上班，下午请半天假。十二点下班，一点回到家，收拾了一下家里的还没来得及收拾的搬家物品，三点半开车去南村万博地铁站接他们。  
回到家后，去市场买菜回来做饭，爸妈也从家里带来了一些海鲜。  
六点樱同学下班，骑小电动去接。  
晚上没什么时候无聊，就想着大家一起出去剪头发，谁知关门后忘记带钥匙，回不去了。  
在小区附近转悠一圈后，决定开车去房东家里拿。距离十来公里，开车半小时。拿了钥匙后在附近找到一个美发护理店，也可以剪头发，就是太慢了，剪完都十一点多了。不过这个toney剪发是真认真，挺满意。  
开车再回到新造，已是十二点，在镇上吃宵夜，猪杂粥和炒河粉，都挺不错，猪杂粥68/斤，一锅。

<!-- node 2026.01.10 周六 --> 
早上十点钟开车出门，去西塱市场买菜。  
- 八九个大闸蟹，17元/个
- 一只走地鸡
- 其他菜
- 想买猪肚，没找到新鲜的
- 水果（柿子、菠萝蜜、沙糖桔）
- 面包、红豆糕

回到家已是两点，爸妈开始做饭。叫上雯瑶+她哥，刚好在广州；舒婷她姐+姐夫；  
大家吃完饭之后坐着聊天了一会儿。五点半我就跟樱同学开车出门去东莞吃喜酒了。  
今天是姚罗璇+李静雯的婚礼，小小代表appoen的这些前同事朋友到场祝福这对新人。  
吃完酒席晚上回到家已是十一点。池湘晚上七点多到，等我回到之后再回去。

<!-- node 2026.01.11 --> 
一起去看了两个家具城，都没什么好看的，不是预期，甚至比不上电城的如意家私。  
中午两点回去后，池湘跟她男朋友再过来，爸妈在小区市场买到了新鲜猪肚，做午饭吃。  
买了七点钟的大巴车票，开车载爸妈去南村万博上车。

<!-- node 2026.01.14 周三 -->

又恢复每周三的周会了😢

<!-- node 2026.01.15 -->
