# 行程大致计划：
- 周五晚上下班，吃完饭后，直接开车前往韶关市区，住一晚
- 第二天早上驱车前往丹霞山景区（提前预订好当前晚上的名宿）
- 直接开车进去景区，到达名宿，把车和行李放到名宿安置
- 吃午饭
- 第一天游玩
- 吃晚饭
- 第二天游玩
- 傍晚晚上驱车回广州
- 广州住宿

# 丹霞山周末详细行程计划（广州出发）
## 一、前期准备
### 1. 票务与住宿
- 门票：提前购买丹霞山全票（100元/人，48小时有效，含景区免费接驳车），电子票可直接刷码入园，无需换票。
- 住宿：
  - 周五晚：韶关市区中心或G106国道附近酒店（推荐韶关风度华美达广场酒店、韶关莱斯大酒店），方便次日早起出发。
  - 周六晚：丹霞山景区内民宿（优先瑶塘村：近丹霞山博物馆，主题民宿集中；断石村：近阳元山、水上丹霞码头），提前与民宿沟通停车事宜（景区内民宿多提供免费停车）。

### 2. 必备物资
- 穿戴：舒适登山鞋（防滑款）、速干衣裤、防晒帽、墨镜、薄外套（山区早晚温差大）。
- 用品：防晒霜（SPF50+）、驱蚊液、饮用水（每人至少2L）、能量零食（巧克力、坚果、牛肉干）、登山杖（可选，助力陡峭路段）、薄手套（抓铁索用）、雨具/防滑鞋套（雨天备用）。
- 其他：身份证、驾驶证、车辆行驶证、少量现金（景区部分小店可能无移动支付）、手机充电宝。

## 二、详细行程安排
### 周五：广州→韶关市区（住宿：韶关市区）
| 时间          | 活动内容          | 具体安排                                                                 |
|---------------|-------------------|--------------------------------------------------------------------------|
| 18:00-18:30   | 下班简餐          | 广州出发地附近吃快餐/简餐（如麦当劳、真功夫或本地简餐店），避免耗时过久，快速补充能量。 |
| 18:30-22:00   | 自驾前往韶关市区  | 导航优先选择广乐高速（G4W3），全程约230公里，行驶2.5-3小时（隧道少、路况稳定）；途中用粤省事APP查看实时路况，避开拥堵路段。 |
| 22:00后       | 入住休整          | 抵达酒店办理入住，卸下行李后尽早休息，避免熬夜，为次日丹霞山游玩储备体力。 |

### 周六：韶关市区→丹霞山（住宿：景区内民宿）
| 时间          | 活动内容          | 具体安排                                                                 |
|---------------|-------------------|--------------------------------------------------------------------------|
| 7:30-8:00     | 韶关市区出发      | 酒店吃早餐（或顺路在路边早餐店买包子、豆浆、油条等便携早餐），沿G106国道自驾前往丹霞山，全程50公里，约1小时可达。 |
| 8:30-9:00     | 安置行李          | 开车进入丹霞山景区（景区入口需查验门票），前往预订的瑶塘村/断石村民宿，停车并放下行李；咨询民宿老板当日景区接驳车运营时间（景区免费接驳车串联长老峰、阳元石、水上丹霞等核心景点，约15分钟一班）。 |
| 9:00-10:00    | 吃午饭            | 在民宿周边的本地餐馆用餐，推荐点酸笋炒肉、清炒鲜笋、客家酿豆腐等开胃小份菜，避免空腹登山，也勿吃得过饱影响行动。 |
| 10:00-12:30   | 游玩阳元石景区    | 步行或乘接驳车到阳元石验票口，走经典体验路线：阳元石观景台（近距离观赏阳元石奇特造型）→拜阳台→云崖栈道→嘉遁亭→细美寨（古村落遗址）→九九天梯（陡峭路段，需抓牢铁索扶手），最后返回验票口，全程约2.5小时，登顶后可俯瞰丹霞群峰全景。 |
| 12:30-13:30   | 午休补给          | 返回民宿附近餐馆吃午饭（推荐尝试丹霞山特色野菜），饭后在民宿短暂休息1小时，避开正午高温（山区正午紫外线强，易疲劳）。 |
| 13:30-16:30   | 体验水上丹霞      | 乘景区接驳车到水上丹霞码头，购买游船票（约90元/人），乘坐游船沿锦江游览，全程50分钟左右，沿途可观赏赤壁丹崖、群象出山、金龟朝圣等经典景观，船上有讲解员介绍地质成因和民间传说；游船结束后，可在码头周边步道散步，欣赏锦江与丹山相映的山水画卷。 |
| 16:30-18:00   | 自由活动          | 体力充沛可选：前往阳元石景区的玄机台（丹霞山最佳日落观赏点之一），等待日落美景，拍照留念；偏爱轻松可选：逛丹霞山博物馆（免费开放，17:00闭馆），通过展品和多媒体了解丹霞地貌形成历史、动植物资源等。 |
| 18:30-20:00   | 享用晚餐          | 在民宿附近的农家乐用餐，必点特色菜：爆炒山坑螺（肉质鲜美）、丹霞走地鸡（山林散养，肉质紧实）、酸笋田螺煲、清蒸锦江鱼，搭配本地米酒或凉茶；用餐时可和老板确认次日观日出的最佳时间（通常6:00-6:30，需结合季节调整）和路线。 |
| 20:00后       | 夜游休整          | 可选体验：《万古丹霞》山水实景光影秀（每晚20:00-21:00，需提前购票，以自然山水为幕布，灯光演绎丹霞历史）；若偏爱安静，可在民宿周边散步，呼吸山间新鲜空气，之后早点休息，为次日长老峰游玩养精蓄锐。 |

|---------------|-------------------|--------------------------------------------------------------------------|

| 11:30-12:30   | 返程民宿吃午饭    | 乘接驳车返回民宿附近，吃顿丰盛的午饭（补充体力），饭后收拾行李，办理民宿退房，开车准备离开景区。 |
| 13:00-16:30   | 自驾返回广州      | 从丹霞山景区出发，经106国道上广乐高速返回广州，全程约230公里，行驶2.5-3.5小时；途中在服务区（如源潭服务区、英德服务区）休息15-20分钟，下车活动筋骨，避免疲劳驾驶，可购买当地特产（如英德红茶、韶关马蹄）。 |
| 16:30后       | 抵达广州休整      | 抵达广州后，前往预订的酒店办理入住，放下行李；若时间充裕，可在酒店周边吃晚餐（推荐清淡饮食，缓解游玩疲惫），之后好好休息，为周一工作/学习调整状态。 |

## 三、温馨提示
1. 门票与接驳车：丹霞山门票48小时内有效，游玩时务必随身携带（电子票存好截图），进出景点、乘坐接驳车需查验；接驳车运营时间约7:00-19:00，合理规划行程，避免错过末班车。
2. 安全注意：九九天梯、丹梯铁索等路段陡峭，雨天路滑，老人、小孩或体能较弱者建议绕行平缓路线；登山时遵循“慢走不看景，看景不走道”，抓牢扶手，切勿翻越护栏。
3. 饮食与补给：景区内补给点较少且价格偏高，登山时务必带足饮用水和能量零食；垃圾随身带走，保护丹霞山自然环境。
4. 自驾须知：景区内道路狭窄，限速30km/h，注意避让行人与接驳车；周末景区车流量大，若民宿停车位紧张，可停在景区入口附近的公共停车场，再乘接驳车进入。
5. 天气参考：丹霞山山区天气多变，出发前查看天气预报，雨天建议取消陡峭路段游览，选择水上丹霞、博物馆等室内/平缓景点。
